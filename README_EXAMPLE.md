系統已設定為自動切換：

## 查看範例資料

1. 啟動網頁伺服器（如果尚未啟動）：
```bash
npx http-server . -p 8080 -o
```

2. 在瀏覽器中訪問 `http://localhost:8080`

3. 您應該會看到：
   - 2 個環境（資轉驗證環境、平測切轉環境）
   - 每個環境有不同顏色的標識
   - 月曆中顯示對應日期的任務
   - 可以點擊日期查看詳細資訊

## 自訂您的 Excel 檔案

您可以參考範例檔案的格式，建立自己的 Excel 檔案：

1. 開啟範例檔案查看格式
2. 複製格式到您的新檔案
3. 填入您的實際資料
4. 將檔案命名為 `藍圖之對應時程環境規劃_v0.1.xlsx` 並放在 `resource/` 目錄下

## Excel 欄位要求

最少需要包含以下欄位（欄位名稱可包含關鍵字）：

- **環境名稱**：包含「環境」關鍵字
- **工作內容**：包含「工作」或「任務」關鍵字
- **開始日期**：包含「開始」或「日期」關鍵字

其他欄位（環境目的、結束日期、狀態等）為可選。

詳細格式說明請參考 `EXCEL_FORMAT.md`
